# Dj DK

## Authors
Daniel Kosykh

## Overview
This is my personal DJ website project to assist with outreach, present my mixes, and present my tracks of the day with the help of a Mongo backend. Originally built as a 2 day sprint, this project will be a continued work in progress and marketing tool for for my hobby as a DJ.

## Technologies Used

- React
- Node
- Express
- Axios
- Styled-Components
- Mongo
- Mongoose
- AWS (EC2)
- Webpack

## Launch

To run, this requires to npm install and a configuration of the nodemailer config file. This config file is necessary for users to be able to send an email to the configured email address.
